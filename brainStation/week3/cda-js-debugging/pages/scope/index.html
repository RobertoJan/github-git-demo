Scope

Some terminology

Scope
current context of execution. The context in which values and expressions are visible or can be referenced.
Memory
information stored for immediate use by the computer. When used as a quantity, refers to the capacity provided by the physical entity for memory in the computer.
Object
an object is a collection of related data and/or functionality.
